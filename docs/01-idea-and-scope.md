# Sa7i / صاحي — الفكرة والنطاق

## الفكرة

تطبيق بسيط جداً بين الأصدقاء:

1. المستخدم يدخل باسم مستخدم وكلمة مرور.
2. أول مرة ما يكون عنده أصدقاء.
3. يضيف شخص باسم المستخدم.
4. الطرف الثاني يقبل الطلب.
5. عند اختيار الصديق تظهر صفحة فيها زر واحد:
   - إذا ما فيه تنبيه وارد: `صاحي ؟`
   - إذا وصل تنبيه من نفس الشخص: الرد الوحيد `صاحي..`
6. الطرف الثاني يسمع صوت تنبيه ويشوف إشعار داخل التطبيق.

## الهدف من النسخة الأولى

هذه نسخة تعليمية/Prototype لتجربة:

- Supabase Auth لإدارة المستخدمين.
- Supabase Postgres للجداول.
- Supabase Realtime للتنبيهات اللحظية.
- Next.js + React للواجهة.
- Vercel للنشر مع SSL.
- PWA عشان يشتغل كتطبيق من الجوال.

## حدود النسخة الأولى

- التنبيه الصوتي مضمون عندما التطبيق مفتوح أو PWA شغال.
- إشعارات الخلفية الحقيقية Push Notifications تحتاج مرحلة ثانية باستخدام Web Push/VAPID أو Supabase Edge Function.
- تسجيل الدخول يستخدم username في الواجهة، لكن Supabase Auth يحتاج email، لذلك نحول داخلياً `username` إلى بريد محلي مثل `username@sa7i.local`.
